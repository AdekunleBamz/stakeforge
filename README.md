# StakeForge

NFT Staking Platform on Base - Stake your NFTs to earn rewards.

## Overview

StakeForge is a decentralized NFT staking platform that allows users to stake their NFTs and earn ERC20 reward tokens over time. Built on Base for low gas fees and fast transactions.
